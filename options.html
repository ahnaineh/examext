Gemini API Key Setup

How to get a Gemini API Key

  1. Go to Google AI Developer Center
  2. Sign in with your Google account
  3. Navigate to the API section
  4. Create a new API key (it should start with "AI-")
  5. Copy and paste the API key above

How to use this extension

After saving your API key, press Alt+Shift+S on any webpage to activate the screenshot tool. Select an area of the screen to analyze with Gemini.
